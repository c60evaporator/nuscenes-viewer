# 推論可視化機能

- "Model"という名前のタブでAIモデルを登録（ここで登録したモデルは後ほど自動アノテーション機能で使用するAIモデルの管理も兼ねる）し、"Inference"という名前のタブでAIモデルの推論可視化機能を提供
- 3D Object Detection、E2E（UniADのようなBEV型を想定し、中間出力も含めて可視化する）の2種類のタスクの可視化を想定。将来的にはVLAモデル等も対象としたいが、今は上記2種類のみでOK

## Inference画面の仕様

- ベースはSample画面のデザイン
- Sample画面右ペイン一番下に”Inference”というボタンを追加し、
- 左ペインはSample画面と同様
- 右ペインでモデル選択。最上部は”Data type”フィルタ（CAMERA or CAMERA+LiDAR）。デフォルトはconfigで指定できるようにする（よく使う方をメインにしたいため）
- その下に”Task”フィルタ（3D Object Detection or E2E）。デフォルトはconfigで指定できるようにする（よく使う方をメインにしたいため）
- その下に”Model”フィルタ。登録されたData type＆Taskと適合するモデル一覧が表示される（3D Object DetectionにはBBox出力可能なE2Eモデルも含む）。デフォルトは何も選択されていない。左ペインでSampleが選択された状態でこのフィルタを選択するとモデル推論が走り、中央ペインにTaskに応じて以下の推論結果が可視化表示される
    - Data type = CAMERA+LiDAR and Task = 3D Object Detectionのとき: Sample画面の中央ペインと同じ構成でセンサデータを表示するが、RADAR_FRONTのみBEV特徴に変える。BEV特徴以外の画像にはアノテーションのBBoxを緑色、推論BBoxを濃オレンジで重ねて表示する
    - Data type = CAMERA and Task = 3D Object Detectionのとき: 以下2パターンを切り替える
        - Sample画面の中央ペインと同じ構成でセンサデータを表示するが、RADAR_FRONTをBEV特徴に変える。BEV特徴以外の画像にはアノテーションのBBoxを緑色、推論BBoxを濃オレンジで重ねて表示する（LiDARは推論に利用できないが、水平位置確認に有用なので表示しておく）。
        - カメラ6台を横3×縦2構成でdivレイアウトし、div中の上側と下側に同じ画像を表示。上側にアノテーションのBBoxを緑色、下側に推論BBoxを濃オレンジで表示
    - Data type = CAMERA+LiDAR and Task = E2Eのとき: Sample画面の中央ペインとほぼ同じ構成でセンサデータを表示するが、RADAR_FRONTのみBEV特徴に変える。正解軌跡を緑色、推論軌跡をオレンジで表示する（中間出力は要検討）
    - Data type = CAMERA and Task = E2Eのとき: 以下2パターンを切り替える
        - Sample画面の中央ペインと同じ構成でセンサデータを表示するが、RADAR_FRONTをBEV特徴に変える。BEV特徴以外の画像には正解軌跡とアノテーションBBoxを緑系の色、推論軌跡と中間出力をオレンジ系の色で重ねて表示する（LiDARは推論に利用できないが、水平位置確認に有用なので表示しておく）。
        - カメラ6台を横3×縦2構成でdivレイアウトし、div中の上側と下側に同じ画像を表示。上側にアノテーションを緑系の色、下側に推論をオレンジ系の色で表示
- “Model”フィルタが選択された状態で左ペインのSample選択を変えると、中央ペインの表示が変更後のSampleに対する推論可視化結果に変わる
- 右ペイン“Model”フィルタの下にモデルの推論パラメータ（しきい値や可視化対象の中間出力等）をモデルの種類に応じて変更できるようプルダウンやチェックボックスで表示（モデル登録で指定したもの）。パラメータを変更すると変更後のパラメータで推論を再実施し、中央ペインに表示する可視化も更新する
- Task = E2Eのとき、中央ペインに表示する中間出力は、物体検出、Mapセグメンテーション、周囲車両の軌跡予測、言語出力（VLA向け）の中から使用可能なもの（モデル登録時に指定）をチェックボックスとして下の欄に表示。チェックボックスを変更すると中央ペインに表示される中間出力も変更する
- Task = 3D Object Detectionのとき、その下にアノテーションごとの推論位置や指標（PoseやIoU等）をリスト表示。リストをクリックすると該当する正解・アノテーションBBoxをハイライト
- その下に推論に対する推論時間等のメトリクスを表示（RADAR_FRONT位置でも良いかも）
- 右ペイン一番下に”Compare models with the Scene”ボタンを設置。押すと別ウィンドウが開き、Data type＆Taskフィルタに適合したモデルをチェックボックスで選択、”Compare Models”ボタンを押すとScene内の全てのデータに全てのモデルの推論を実施し、各データに対する推論成否や、Scene全体から計算した各種性能評価指標を一覧表示（別ウィンドウなので、アプリ本体の推論可視化と見比べるユースケースを想定）
